wisdom_modules/common/src/lib/http-context/use-loader.ts

USE_LOADER
Type : miscellaneous
Default value : new HttpContextToken<string | boolean>(() => false)

HttpContextToken used to determine if loader should be shown. A given string will be run through the translator and therefore may get translated.

Useful for longer loading times.

Defaults to false.
